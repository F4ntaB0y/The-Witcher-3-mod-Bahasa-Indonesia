# Mod Bahasa Indonesia - The Witcher 3: Wild Hunt

Mod ini menambahkan lokalisasi Bahasa Indonesia untuk game *The Witcher 3: Wild Hunt*.

**Pembuat:** Arya Pangestu (Aestu)

---

## Cara Instalasi

1. Unduh berkas mod.
2. Ekstrak berkas mod yang telah diunduh.
3. Salin folder `Indonesia` (atau isi dari folder mods) ke dalam direktori instalasi game Anda (folder `mods`).
4. Jalankan game dan nikmati terjemahannya!
